Начало работы

Три шага перед настройкой софтов.

  1. Скачайте лаунчер через Telegram-бота @ProfitonSupport_bot Сохраните файл сразу — через 20 минут он автоматически удаляется из бота. Через него же подтверждается вход в лаунчер и включение/выключение софтов.
  2. Подпишитесь на новостной канал @Profiton_News
  3. Присоединяйтесь к чату сообщества @Profiton_Chat После вступления напишите в поддержку для снятия ограничения на отправку сообщений.
